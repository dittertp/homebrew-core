# Dittertp Core

## How do I install these formulae?

`brew install dittertp/core/<formula>`

Or `brew tap dittertp/core` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "dittertp/core"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
